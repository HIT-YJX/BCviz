Index Similarity
========

Please finish the experiment "2.IndexConstructionTime" first.

Here, we can get the following figures and datas of the paper.

Fig. 8
Run the code "Fig.8.py", and input the absolute path of the BCviz index directory "~/BCviz/Index-results/", for example, "D:/BCviz/Index-results/".
Then you can get the figures. Two files "Marvel-IndexVisulization.jpg" and "WikiPedia-IndexVisulization.jpg" will be generated in current directory.

Similarity
Run the code "Kendalltua.py", and input some parameters:
1. input the dataset name, 
2. input the absolute path of the BCviz index directory "~/BCviz/Index-results/", for example, "D:/BCviz/Index-results/".
Then, it will output the similarility results to the console.
